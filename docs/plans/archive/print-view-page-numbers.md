# 印刷用ページ: 印刷時のみページ番号を表示

## 目的・背景

管理画面の印刷用表示（printView.ts）で複数ページの教材を印刷すると、
ページ番号が無く紙の並び順が分からなくなる。印刷時のみページ番号を表示する。

## 対応方針

- `renderPrintPageHtml` の `@page` ルールにマージンボックス
  `@bottom-center { content: counter(page) " / " counter(pages); }` を追加する
- `@page` はページドメディア（印刷/PDF）にのみ適用されるため、
  画面表示には出ない（「印刷時だけ」の要件を CSS の仕組みで満たす）
- Chrome 131+ が CSS マージンボックスに対応（手元は 149）。未対応ブラウザでは
  単にページ番号が出ないだけで、本文の印刷には影響しない

## 影響範囲

- backend の `src/printView.ts`（共通テンプレートのため全印刷ページに反映）と
  `test/printView.test.ts` のみ

## テスト方針

- 単体テスト: 生成 HTML にページ番号のマージンボックス CSS が含まれること
- E2E: 複数ページになる Document 訳（#4）を print-to-pdf し、各ページ下部に
  「n / 総ページ数」が出ること、画面表示には出ないことを確認
